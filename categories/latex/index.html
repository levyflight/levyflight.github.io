Create Lyx layout for IIT Madras thesis

Mon Mar 27, 2017 Categories: Latex . Tags: Lyx .

The latex template for IIT Madras PhD thesis can be found here. This is enough to write your thesis in Latex. But if you prefer to write your thesis in LyX then you need a layout for IITM thesis class. To create a layout you need to follow the steps below (suggested here): Download latex template from here. Untar the tar file tar xzf iitm_thesis.tar.gz Open iitmdiss.cls file and remove compat2 in line number 74.
